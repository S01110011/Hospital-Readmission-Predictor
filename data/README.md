# Dados

Este diretorio e separado em:

- `raw/`: dados brutos ou sinteticos antes do processamento.
- `processed/`: dados limpos e prontos para modelagem.

Dados reais de pacientes nao devem ser versionados. Em contexto real, use controles de acesso, criptografia, anonimizacao/pseudonimizacao, logs de acesso e governanca de retencao.
